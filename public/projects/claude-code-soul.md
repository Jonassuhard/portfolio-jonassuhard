# claude-code-soul - pack de configuration Claude Code

Type : config claude code · public.

Période : 2026.

Rôle : Auteur.

Stack : Claude Code CLI, Shell (hooks), Markdown, macOS Keychain + Touch ID, gitleaks.

Statut : Public (MIT).

En bref : Pack de configuration open source pour Claude Code CLI : un soul.md (identité/ton/refus) plus skills, agents, hooks et rules, pour donner une personnalité dosée au CLI.

## Problème

Séparer l'âme de l'assistant (ton, valeurs, refus) de la config technique, et packager un outillage Claude Code réutilisable et publiable sans fuite de données.

## Ce que ça montre

- Système de configuration en couches, documenté et installable via script.
- Sécurité by design : Touch ID, secrets via Keychain, 0 PII.
- Publication open source propre (MIT) avec crédits des packs tiers.

## Limites

- Config opinionnée : à adapter avant réutilisation.
- Certaines briques tierces exclues du repo.
